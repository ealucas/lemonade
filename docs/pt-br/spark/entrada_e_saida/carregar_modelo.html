Carregar Modelo

Carrega um modelo previamente gerado pelo LEMONADE e salvo usando a operação Salvar modelo. O modelo carregado pode ser aplicado em um novo conjunto de dados, sendo reutilizado.

Exemplo de Utilização

Objetivo: Carregar um modelo salvo. No exemplo apresentado, será utilizada a base de dados íris. O modelo será carregado, aplicado a base íris e avaliado.
Base de Dados: Íris

  1. Adicione uma base de dados por meio da operação Ler dados.

  2. Na operação Converter categórico para numérico, selecione “class” no campo Atributos. Preencha “class_label” no campo Nome para novos atributos indexados.

  3. Adicione a operação Aplicar modelo, selecione “petal_length”, “petal_width”, “sepal_length” e “sepal_width” no campo Atributos previsores e preencha “prediction” no campo Nome do novo atributo.

  4. Adicione a operação Carregar modelo e selecione o modelo que será utilizado no fluxo.

  5. Adicione a operação [Avaliar Modelo] selecione “prediction” no campo Atributo usado para predição, “class_label” no Atributo usado como label, “Acurácia” em Métrica usada para avaliação.

  6. Execute o fluxo e visualize o resultado.
