Exemplo de Utilização

Objetivo: observar a variação das da temperatura no Brasil.

Base de Dados: Temperatura Global por País

  1. Adicione uma base de dados por meio da operação Ler dados.

  2. Adicione a operação Agrupar linhas por função. Na opção filtro, selecione o atributo “Country”, função “Equals to =” e preencha o Valor do atributo como “Brazil”.

  3. Na operação Gráfico de área selecione “dt” no Atributo para eixo X e “AverageTemperature” no Atributos para eixo Y.

  4. Execute o fluxo e visualize o resultado.
    Observe que todas as classes possuem 50 instâncias.
